# Sword Art

## Описание

*На данный момент: простая игра с деревом анимаций действий игрока.*

## Фичи

<ul>
  <li>Low-Poly стилизация</li>
  
  <li>Машина состояний анимаций через Blend Tree</li>
</ul>

## Технологии

<ul>
  <li>Unity 3D 2022.3.47f1</li>

  <li>C# v.8</li>
</ul>

## Инструменты

<ul>
  <li>Terrain Tool</li>
  
  <li>Animator with blend tree</li>
</ul>
