# Sword Art

## Описание

*На данный момент: простая игра с деревом анимаций действий игрока.*

## Фичи

Low-Poly стилизация

Машина состояний анимаций через Blend Tree

## Технологии

Unity 3D 2022.3.47f1

C# v.8

## Инструменты

Terrain Tool

Animator with blend tree
